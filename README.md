
# Chain of Responsibility Pattern

```windows cmd
cd C:\ITSA\designpatterns\src\main\java
javac itsa\patterns\CORExample.java

```

```windows cmd
cd C:\ITSA\designpatterns

apache-maven-3.6.3\bin\mvn test -Dtest=CORExampleTest
```
